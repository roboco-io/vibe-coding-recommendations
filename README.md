# Vibe Coding Recommendations

AI 도구를 활용한 개발 생산성 극대화를 위한 종합 가이드

**버전 2026.1**

[English](README.en.md) | [日本語](README.ja.md)

## 개요

이 프로젝트는 "바이브 코딩"에 대한 표준 가이드를 제공합니다. 설계와 구현부터 검증, 문서화, 운영에 이르기까지 소프트웨어 개발 라이프사이클 전체에서 AI 도구를 활용하는 포괄적인 접근법입니다.

본 가이드는 2024년 11월부터 2026년 1월까지 축적된 실전 경험을 바탕으로, 이미 확립된 프로세스를 가진 사람보다는 바이브 코딩 여정을 막 시작하는 개발자를 대상으로 작성되었습니다.

## 핵심 개념

- **생산성 공식**: `바이브 코딩 생산성 = 개발자 역량 × AI 도구와 프로세스의 생산성`
- **핵심 원칙**: "AI를 믿지 말라! 사람도 믿지 말라! 신뢰할 수 있는 프로세스를 만들어라!"
- **YOLO 모드**: 자동화된 테스트, 리뷰, 백업 루틴에 의해 뒷받침되는 빠른 실행
- **ZDR (Zero Data Retention)**: 코드가 AI 학습에 사용되지 않도록 하는 필수 모드
