# Vibe Coding Recommendations

Claude Codeなどのツールを使用した、AIアシスト開発手法による生産性最大化のための包括的なガイド

**バージョン 2026.1**

## 概要

このプロジェクトは「vibe coding」の標準ガイドを提供します。設計・実装から検証・ドキュメント化・運用まで、ソフトウェア開発ライフサイクル全体でAIツールを活用する包括的なアプローチです。

本ガイドは2024年11月から2026年1月にかけて蓄積された実践経験に基づいており、既に確立されたプロセスを持つ方よりも、vibe codingを始めたばかりの開発者を対象としています。

## 主要コンセプト

- **生産性の公式**: `Vibe Coding生産性 = 開発者の能力 × AIツール・プロセスの生産性`
- **核心原則**: 「AIを信じるな!人も信じるな!信頼できるプロセスを構築せよ!」
- **YOLOモード**: 自動化されたテスト・レビュー・バックアップルーチンに支えられた高速実行
- **ZDR (Zero Data Retention)**: コードがAIトレーニングに使用されることを防ぐための必須モード

## 技術スタック

- **バックエンド**: Go

- **マルチプラットフォーム**: Flutter/Dart
- **AIツール**: Claude/Claude Code
- **ドキュメント**: Markdown, Marp
